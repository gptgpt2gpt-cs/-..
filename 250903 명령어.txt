<신청서 스캔본 업로드 프로그램 만들기>

9/3 다운 프로그램이 아닌 사이트로 만들기 위해 초기화 후 새로운 명령어 입력

명령어)


파일을 넣으면 아래 방법과 조건으로 @https://todo.ktc.re.kr/ 사이트에 업로드해주는 사이트 만들어줘. 

<방법>
1. 크롬을 열어서 @https://todo.ktc.re.kr/ 사이트 접속
2. 아이디 입력란에 'lsa211'기재하고, 비밀번호 입력란에 'ktc1234' 기재 후 로그인 버튼 클릭 또는 enter
3. 좌측 상단 '시험검사' 메뉴 클릭
4. 좌측 '현황관리' 클릭
5. '현황관리' 하단 '전체현황조회' 클릭
6. 우측 '조회'버튼 하단 '초기화' 버튼 클릭
7. '검색조건' 입력란에 아래 조건으로 기재 
 -조건: 업로드 사이트에 넣은 상위 첫번째 파일명의 첫 글자부터 '-'기호 뒤 숫자 5글자까지 (예시: 업로드 된 파일명이 'CXA2025-00982.pdf' 라면 'CXA2025-00982'를 기재)
8. 우측 '조회' 버튼 클릭
9. 좌측 하단 '시험분류' 하단의 글씨 클릭하여 '시험진행 상세정보' 화면 접속
10. '첨부자료' 우측 '조회' 버튼 클릭
11. '첨부자료조회' 화면 창의 '첨부파일'란에 방법 7번에 선택했던 파일 이동 후 '저장'버튼 클릭.
12. '첨부자료조회'화면 우측 상단의 엑스모양의 창닫기 버튼 클릭
13. 우측 상단의 '목록'버튼 클릭
14. 방법 6번~13번 반복

<조건>
1. 사이트 디자인
 1) 사이트의 배경 및 디자인 색상은 @http://gw.ktc.re.kr/gw/uat/uia/egovLoginUsr.do 사이트의 배경 및 다자인 색상 차용
 2) 가독성이 좋아야 함.
 3) 기본 글씨 색상은 검정색
2. 업로드 사이트에 넣은 모든 파일의 업로드 행위가 끝나면, 결과에 따라 해당 문구의 팝업창 띄우기
 1) 결과 1 : 전체 업로드 성공 시 
    - 문구 : 전체 파일 업로드가 완료 되었습니다. (총 x건 중 x건)
 2) 결과 2 : 일부 업로드 성공 시 
    - 문구 : 일부 파일의 업로드가 실패하였습니다. (총 x건 중 x건 성공, x건 실패)
 3) 결과 3 : 전체 실패 시 
    - 문구 : 파일의 업로드가 실패하였습니다. (총 x건 중 x건 실패)

2. 사이트 작동 중 오류 발생 시, 아래 문구와 오류 사유 기재하여 팝업창 띄우기
 - 문구 :오류 발생. 고객가치지원실 이신애 책임연구원에게 오류 화면 캡쳐하여 메신저로 문의 주시기 바랍니다.
 - 문구를 상위, 오류 사유를 그 아래에 구분하여 표기

3. 사이트의 구동 속도가 느리지 않도록, 빠르게 최적화 시켜줘
